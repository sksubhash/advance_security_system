first clone the repo from GitHub
for cloning write : 
>git clone https://github.com/sksubhash/advance_security_system.git


now install the requirement from requriment.txt    
Go to Your Project Directory in my case C:\Users\subhash\PycharmProjects\AdvanceSecuritySystem is my project Directory . Now Install

>pip install requriment.txt 


create a sql database in your xampp/wampp

>Python manage.py migrate


Now Run Server Using 
>python manage.py runserver
